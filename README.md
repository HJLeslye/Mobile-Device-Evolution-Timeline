# Timeline: Evolution of Mobile Devices

An interactive, arcade-inspired timeline documenting the most significant milestones in mobile telephony—from the Motorola DynaTAC 8000X to modern AI-driven foldable devices.

This project was developed with a **80s Retro-Arcade** aesthetic, featuring advanced CSS animations and a dynamic reference system in **APA 7th Edition** format.

## Key Features

* **Retro-Arcade Interface:** Animated grid background and neon color palette.
* **APA 7th Edition Tooltips:** Hover over references to see full academic citations instantly.
* **Responsive Design:** Fully optimized for mobile and desktop browsers.
* **Retro HUD:** Gaming-inspired header displaying "Score" and "Lives" for an immersive experience.

## Built With

* **HTML5:** Semantic structure for historical levels.
* **CSS3:** Custom animations (keyframes), Flexbox, and CSS Variables.
* **Google Fonts:** Using the 'Press Start 2P' pixel font.

## Project Structure

* `index.html`: Main structure and HUD implementation.
* `style.css`: Styles, neon effects, and energy flow animations.
* `/img`: Visual resources for each mobile device.
